This series of tasks will help you learn the basics of OOP in Python, such as creating classes, attributes, and methods, inheritance, method overrides, using super(), and polymorphism.

Mission 8: Class methods with parameters.

  • 8.1 Add is_used attribute with default value of False to Car class objects and initialize it in __init__ method.
  • 8.2 Add a drive method to the Car class that takes a distance argument, displays the message "Driven {distance} km" for objects of class Car and changes is_used to True.
  • 8.3 Create object yet_another_car of class Car and call the drive method, passing the appropriate distance.
