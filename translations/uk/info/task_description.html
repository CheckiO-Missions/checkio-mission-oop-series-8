8.1. Додай метод drive до класу Car, який приймає аргумент distance і виводить повідомлення "Driven {distance} km", якщо тип distance дорівнює int, а інакше "Wrong 'distance' value type".

8.2. Перепиши метод для класу ElectricCar, щоб успішне повідомлення було "Driven {distance} km on electric motor".

8.3. Ми викличемо метод drive для списку різних автомобілів. Твій код повинен повертати відповідне повідомлення в залежності від типу автомобіля та значення відстані.
